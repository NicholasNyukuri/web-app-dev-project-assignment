# Nextech Academy Website

## Project Overview

Nextech Academy is a responsive educational website developed as part of a Web Application Development project. The website showcases a modern technology learning platform that offers courses in Web Development, Programming, Cybersecurity, Data Science, Artificial Intelligence, and other digital skills.

The project demonstrates the use of HTML5, CSS3, JavaScript, Bootstrap 5, responsive design principles, and Git/GitHub version control.

---

## Features

### Responsive Design
- Built using Bootstrap 5
- Mobile-friendly navigation bar
- Responsive layouts
